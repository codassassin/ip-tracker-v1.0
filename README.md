# ip-tracker-v1.0
This is a ip tracker that can trace back the location of the machine that executes it.

## How to use?
* First:
```
$ git clone https://github.com/codassassin/ip-tracker-v1.0.git
```
* Run the IpTrackerv1.0.py file:
```
$ python IpTrackerv1.0.py
```
